# DARLING
![img](https://cdn.jsdelivr.net/gh/obentnet/DARLING@1.0.1/images/theme_logo.png)
## 使用
1.请前往[releases](https://github.com/obentnet/DARLING/releases/)页面下载最新发行版，确保问题比较少。

2.上传主题到主题文件夹内。

3.登录后台,启用DARLING主题即可。
